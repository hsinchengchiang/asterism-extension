🛠️ 开发者模式安装 (Beta Manual Installation)

由于插件目前正在进行深入审核，内测用户请通过以下方式手动加载：

01.下载解压：下载本项目代码并解压。

02.开启模式：在 Chrome 地址栏输入 chrome://extensions/ 并开启右上角的“开发者模式”。

03.加载路径：点击 “加载已解压的扩展程序”，选择解压好的文件夹「chrome_extension」。

-------------------------------------------------------------------------

🚀 快速上手 (Quick Start)

01.账号注册：访问 [asterism.art](https://asterism.art/) 完成基础账号注册。

02.双端登录：请确保在 Web 网页端 和 Chrome 插件端 同时登录同一账号，以激活实时语义同步。

03.即时捕获：在任意网页浏览时，只需对着心动的图片点击 右键 -> Add to Asterism，稍等片刻，该素材将被载入[asterism/canvas](https://asterism.art/canvas)，点击右上角「重新整理」即可查看
