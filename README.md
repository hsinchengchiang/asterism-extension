🚀 快速上手 (Quick Start)

为了确保灵感碎片能完美同步至你的个人星图，请按照以下流程操作：

安装插件：按照下方的 开发者模式安装 步骤完成插件加载。

账号注册：访问 asterism.art 完成基础账号注册。

双端登录：请确保在 Web 网页端 和 Chrome 插件端 同时登录同一账号，以激活实时语义同步。

即时捕获：在任意网页浏览时，只需对着心动的图片点击 右键 -> Add to Asterism，该素材将瞬间被语义引力捕捉并存入你的私人星野。

🛠️ 开发者模式安装 (Beta Manual Installation)
由于插件目前正在进行深入审核，内测用户请通过以下方式手动加载：

下载解压：下载本项目代码并解压。

开启模式：在 Chrome 地址栏输入 chrome://extensions/ 并开启右上角的“开发者模式”。

加载路径：点击 “加载已解压的扩展程序”，选择项目中的 dist 文件夹。
